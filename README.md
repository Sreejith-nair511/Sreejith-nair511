## About

Game developer and AI engineer with a focus on two things: crafting interactive worlds that feel alive, and building systems that make them smarter.

Founder of **JuGaad.exe** (trading as ZreeGames) — an indie game studio with 25+ titles published on itch.io. Active collaborator on ISRO space tech. National champion at E-MINDS 2024, placing 1st out of 500+ teams in a 28-hour marathon. Top contributor in GirlScript Summer of Code with 200+ merged PRs.

Currently building a Steam RPG with ML-Agents-powered NPCs, a real-time satellite tracker for ISRO, and an adaptive AI companion system for games.
